Send us your own custom cards :

Please make sure your card is vertical with a 2/3 ratio. 500x750 pixels is a good size.
Also make sure you have all the rights on the image ^^
Then send at jules.fouchy@ntymail.com either :
  • Your image in png or jpeg format
  • For the code artists : you can also send us a p5 script ! It will be run each time the card is drawn and if it involves some randomness the card will be different each time !
  • You can also send us a fragment shader written in glsl ;) (feature incoming)
